## 🚀 如何运行

### 移动应用（推荐）

1. **安装依赖：**
   ```bash
   cd yanbao-v2.2.0
   pnpm install
   ```

2. **启动开发服务器：**
   ```bash
   npx expo start
   ```

3. **扫描二维码：**
   - iOS：使用 Expo Go 应用扫描
   - Android：使用 Expo Go 应用扫描

### Web 预览（备选）

```bash
npx expo start --web
```
