# yanbao AI v2.2.0 商用级 UI 设计系统

## 📐 设计概述

基于用户提供的 22 张设计参考图，本设计系统为 yanbao-imaging-studio 进行完全重构，融合库洛米品牌元素，达到商用软件质感标准。

## ✨ 视觉效果

### 玻璃态设计（Glassmorphism）

```
背景模糊：blur(10px)
背景透明度：rgba(255, 255, 255, 0.1)
边框：1px solid rgba(255, 255, 255, 0.2)
阴影：0 8px 32px rgba(0, 0, 0, 0.1)
```

### 发光效果（Glow）

```
弱发光：0 0 20px rgba(255, 107, 157, 0.4)
强发光：0 0 30px rgba(255, 107, 157, 0.8)
脉冲：animation: glow-pulse 2s ease-in-out infinite
```

| 深 | 0 16px 32px rgba(0,0,0,0.16) | 最高层级 |

## 📐 响应式设计

### 断点

- **Mobile**：< 640px
- **Tablet**：640px - 1024px

### 适配规则

- 首页：全屏适配
- 拍照：竖屏专用
- 编辑：竖屏为主
