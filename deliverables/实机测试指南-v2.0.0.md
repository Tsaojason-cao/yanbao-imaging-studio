## 📥 下载 APK

### 方法一：通过 Expo 构建页面下载

1. 访问构建页面：
   ```
   https://expo.dev/accounts/tsaojason/projects/yanbao-eas-build/builds/0fab346f-672d-417f-8c3e-0072c4a3ed48
   ```

2. 等待构建完成（状态显示为 "FINISHED"）

3. 点击页面上的 **"Download"** 按钮下载 APK 文件

4. APK 文件名格式：`build-0fab346f-672d-417f-8c3e-0072c4a3ed48.apk`

### 方法二：通过官网下载

1. 访问官网：
   ```
   https://tsaojason-cao.github.io/yanbao-screenshots-showcase/
   ```

2. 滚动到 **"立即下载"** 部分

3. 点击 **"📥 下载 APK"** 按钮

4. 跳转到 Expo 构建页面后，点击 Download 按钮

### 方法三：使用命令行检查构建状态

在项目目录中运行：

```bash
cd /home/ubuntu/yanbao-imaging-studio
./check-build-status.sh
```

该脚本会自动检查构建状态并显示下载链接。

## 📱 安装步骤

### 在 Android 设备上安装

1. **下载 APK 文件**
   - 使用上述任一方法下载 APK 文件到您的 Android 设备

2. **允许安装未知来源应用**
   - 打开 **设置** → **安全** → **未知来源**
   - 或在安装时系统会提示允许安装

3. **安装 APK**
   - 找到下载的 APK 文件（通常在 `Downloads` 文件夹）
   - 点击文件开始安装
   - 按照屏幕提示完成安装

4. **启动应用**
   - 在应用列表中找到 **"雁宝 AI 私人影像工作室"**
   - 点击图标启动应用
