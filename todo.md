# YanBao AI - 工业级UI重构任务清单

## 核心设计理念

### 1. 全屏沉浸式体验
- [ ] 去除所有顶部标题栏和返回按钮
- [ ] 整个界面就是库洛米的世界
- [ ] 背景永远是库洛米或相机预览

### 2. 拟物化轮盘导航（核心）
- [ ] 中心3D库洛米头像（可点击）
- [ ] 周围5个功能按钮（主页/相机/编辑/相册/设置）
- [ ] 金属光泽或发光边缘效果
- [ ] 视觉上要"立起来"（Neumorphism）
- [ ] 点击圆盘图标直接切换功能

### 3. 绝对定位布局系统
- [ ] 不使用Stack Navigator
- [ ] 使用Absolute Positioning
- [ ] 所有功能面板通过轮盘触发弹出
- [ ] 关闭时直接滑动消失
- [ ] 保持背景永远可见

### 4. 深度毛玻璃效果（Glassmorphism）
- [ ] 所有UI组件漂浮在背景上
- [ ] 带阴影和模糊感（Z-Index层级感）
- [ ] 参数面板悬浮在相机取景框之上
- [ ] 深度毛玻璃效果，不是死板色块

### 5. 库洛米元素深度融合
- [ ] 拍照快门：库洛米浮雕头像
- [ ] 调节滑杆：两端用紫色骷髅头点缀
- [ ] 加载动画：库洛米在屏幕中间跑动
- [ ] 统计卡片：左侧配紫色骷髅头小图标
- [ ] 滤镜预览：选中的滤镜带库洛米耳朵边框
