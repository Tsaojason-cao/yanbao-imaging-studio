- 边框颜色：#E879F9（粉紫色）
- 边框宽度：3px
- 发光效果：shadowRadius 15

**4. 使用提示**
- 文字："选择裁剪比例后，拖动图片进行裁剪"
- 颜色：灰色
- 字号：12px
- 位置：裁剪按钮下方

## 🚀 如何在真实设备上查看

### 方法 1：使用 Expo Go（推荐）

1. **下载 Expo Go**
   - iOS: App Store 搜索 "Expo Go"
   - Android: Google Play 搜索 "Expo Go"

2. **启动开发服务器**
   ```bash
   cd yanbao-v2.2.0
   npx expo start
   ```

3. **扫描二维码**
   - 使用 Expo Go 扫描终端显示的二维码
   - 应用会自动加载

4. **查看效果**
   - 浏览所有 6 大模块
   - 验证上述所有 UI 效果
   - 截图保存

### 方法 2：构建 APK/IPA

1. **配置 EAS Build**
   ```bash
   npm install -g eas-cli
   eas login
   eas build:configure
   ```

2. **构建应用**
   ```bash
   # Android
   eas build --platform android --profile preview
   
   # iOS
   eas build --platform ios --profile preview
   ```

3. **安装到设备**
   - 下载生成的 APK/IPA
   - 安装到手机
   - 查看完整效果
