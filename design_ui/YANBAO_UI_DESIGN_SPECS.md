## 📐 7 个模块的完整设计规范

### 模块 1: 星光轮盘首页

**设计特点**：
- 中心库洛米头像（80dp 圆形，粉色背景）
- 8 个功能按钮围绕轮盘（极坐标分布）
- 底部两个大按钮（拍照、相册）
- 星光渐变背景

**交互说明**：
1. 用户打开应用，进入星光轮盘首页
2. 点击轮盘上的任意按钮，进入对应功能模块
3. 底部"拍照"按钮进入原生相机
4. 底部"相册"按钮进入作品集

**View ID 映射**：
- `wheel_container` - 轮盘容器
- `kuromi_avatar` - 库洛米头像

- `btn_camera` - 拍照按钮
- `btn_gallery` - 相册按钮

### 模块 2: 29 维参数编辑器

**设计特点**：
- 3 层卡片式架构（皮相、光影、骨相）
- 每个参数都有 SeekBar 滑块
- 实时数值显示（+100/-100）
- 可展开/折叠的卡片设计

5. 毛孔细致 - 毛孔细节
6. 去噪 - 噪点去除
7. 纹理 - 肌肤纹理
8. 饱和度 - 色彩饱和度
9. 色温 - 色温调整
10. 色调 - 色调调整

**光影（9 维）**：
1. 感光度 - ISO 调整
2. 对比度 - 图像对比度
3. 高光 - 高光区域
4. 阴影 - 阴影区域
5. 黑点 - 黑点调整
6. 白点 - 白点调整
7. 清晰度 - 图像清晰度
8. 晕影 - 边缘晕影
9. 柔焦 - 柔焦效果

**骨相（10 维）**：
1. 脸型调整 - 整体脸型
2. 颧骨高度 - 颧骨位置
3. 下颌线 - 下颌线条
4. 额头宽度 - 额头宽度
5. 鼻型 - 鼻子形状
6. 眼睛大小 - 眼睛大小
7. 唇部丰满 - 唇部丰满度
8. 下巴长度 - 下巴长度
9. 脸部宽度 - 脸部宽度
10. 锐化 - 面部锐化

**交互说明**：
1. 用户打开参数编辑器
2. 选择要调整的分组（皮相、光影、骨相）
3. 点击参数卡片展开详细调整界面
4. 拖动 SeekBar 调整参数值
5. 实时预览效果
6. 保存或重置参数

**View ID 映射**：
- `container_skincare` - 皮相分组容器
- `container_lighting` - 光影分组容器
- `container_bone` - 骨相分组容器
- `param_*` - 各参数对应的 View ID
- `seekbar_*` - 各参数的 SeekBar
- `value_*` - 各参数的数值显示

### 编码规范

**命名规范**：
- View ID：`btn_*`（按钮）、`tv_*`（文字）、`iv_*`（图片）、`sv_*`（SeekBar）
- 资源 ID：`@+id/view_name`
- 颜色引用：`@color/color_name`
- 字符串引用：`@string/string_name`
- 尺寸引用：`@dimen/dimen_name`

**代码风格**：
- 使用 ViewBinding 或 findViewById
- 实现 OnClickListener 处理点击事件
- 使用 RecyclerView 处理列表
- 使用 SeekBar.OnSeekBarChangeListener 处理滑块
