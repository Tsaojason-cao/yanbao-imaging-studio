## 🚀 技术实现

### 框架与技术栈
- **框架**: React Native + Expo
- **语言**: TypeScript
- **组件**: 原生 React Native 组件（View, Text, TouchableOpacity, Image, ScrollView）
- **平台**: Android & iOS（原生移动应用）
- **导航**: Expo Router + Tabs Navigator（8 个 Tab）
