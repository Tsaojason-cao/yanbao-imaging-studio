## 🇨🇳 中国摄影师 (5 位)

### 1. 肖全 (Xiao Quan) - 时代记录者

> “我们这一代人，是伴随着一种理想主义的、英雄主义的、浪漫主义的激情长大的。”

- **风格描述**: 极致的黑白人像，强调眼神光与皮肤的粗粝纹理，赋予照片穿越时间的厚重感与叙事性。他的镜头下，人物的灵魂仿佛被定格。
- **代表色调**: 高对比度黑白、丰富的灰阶、硬朗的光影。

| 参数分类 | 参数项 | 数值 | 备注 |
| :--- | :--- | :--- | :--- |
| **美颜** | 磨皮 (smooth) | 0 | 完全保留皮肤纹理 |
| | 亮眼 (bright) | 20 | 强化眼神光 |
| | 原生肤质保护 | 100 | 确保皮肤细节不丢失 |
| **滤镜** | 饱和度 (saturation) | -100 | 纯黑白 |
| | 对比度 (contrast) | 25 | 增强光影反差 |
| | 亮度 (brightness) | -15 | 压低整体影调，增加深邃感 |
| | 颗粒感 (grain) | 15 | 模拟胶片质感 |
| | 暗角 (vignette) | 15 | 视觉聚焦于中心人物 |

### 2. 孙郡 (Sun Jun) - 新文人画

> “我只是想把我心中的那种感觉给表达出来，那种很宁静的、很东方的、很中国的感觉。”

- **风格描述**: 将摄影与中国工笔画完美融合，画面素雅宁静，意境悠远。色彩饱和度极低，人物肤色如白瓷般剔透，整体呈现出浓郁的中国风韵。
- **代表色调**: 低饱和度、暖调、柔和高光、古典色彩。

| 参数分类 | 参数项 | 数值 | 备注 |
| :--- | :--- | :--- | :--- |
| **美颜** | 磨皮 (smooth) | 15 | 营造细腻肤质 |
| | 红润 (blush) | 15 | 增添一丝古典气色 |
| | 骨相立体 | 10 |  subtle 增强面部结构 |
| **滤镜** | 饱和度 (saturation) | -35 | 营造素雅感 |
| | 色温 (temperature) | 10 | 偏暖调，增加古典韵味 |
| | 高光抑制 | 20 | 避免高光过曝，保持柔和 |
| | 阴影补偿 | 15 | 提亮暗部，展现细节 |

### 3. 陈漫 (Chen Man) - 视觉艺术家

> “我觉得中国当代的美，应该是一种和谐的、有生命力的、积极的、健康的、性感的。”

- **风格描述**: 色彩极其饱和、时尚感强烈、皮肤修饰完美、极具视觉冲击力。她的作品重新定义了中国当代时尚摄影的美学标准。
- **代表色调**: 高饱和度、高对比度、明亮、强烈的色彩碰撞。

| 参数分类 | 参数项 | 数值 | 备注 |
| :--- | :--- | :--- | :--- |
| **美颜** | 磨皮 (smooth) | 40 | 追求完美无瑕的皮肤 |
| | 骨相立体 | 45 | 打造超模般的立体轮廓 |
| | 黑眼圈淡化 | 40 | 确保面部光洁 |
| **滤镜** | 饱和度 (saturation) | 20 | 提升色彩鲜艳度 |
| | 对比度 (contrast) | 15 | 增强视觉冲击力 |
| | 锐度 (sharpness) | 15 | 让画面更清晰、更有力 |
| | 色相偏移 (hueShift) | 5 | 偏向洋红，营造时尚感 |

## 🇯🇵 日本摄影师 (5 位)

### 1. 杉本博司 (Hiroshi Sugimoto) - 禅意长曝

> “我拍摄的是时间的化石。”

- **风格描述**: 极致的长曝光与深刻的禅意，画面呈现出超越现实的宁静与空灵。通过灰度滤镜和精细的影调控制，模拟大画幅银盐相纸的独特质感。
- **代表色调**: 极简黑白、丰富的中间调、柔和的过渡、哲学思辨。

| 参数分类 | 参数项 | 数值 | 备注 |
| :--- | :--- | :--- | :--- |
| **美颜** | 原生肤质保护 | 100 | 完全保留原始质感 |
| **滤镜** | 饱和度 (saturation) | -80 | 接近纯黑白，但保留微弱色调 |
| | 对比度 (contrast) | -20 | 降低反差，营造柔和感 |
| | 亮度 (brightness) | 35 | 模拟过曝效果，画面空灵 |
| | 锐度 (sharpness) | 15 | 保持细节清晰 |
| | 褪色 (fade) | 10 | 增加灰度层次 |

### 2. 蜷川实花 (Mika Ninagawa) - 梦幻花卉

> “对我来说，摄影就像是呼吸一样自然。”

- **风格描述**: 浓烈、饱和、梦幻，以极度绚烂的色彩和繁盛的花卉元素著称。她的作品充满了生命力与幻想色彩，将现实世界装点得如梦似幻。
- **代表色调**: 高饱和度、高亮度、强烈的色彩（尤其是粉、红、蓝）、梦幻感。

| 参数分类 | 参数项 | 数值 | 备注 |
| :--- | :--- | :--- | :--- |
| **美颜** | 磨皮 (smooth) | 30 | 打造娃娃般的肌肤 |
| | 红润 (blush) | 20 | 增添梦幻气色 |
| | 骨相立体 | 25 | 增强面部轮廓 |
| **滤镜** | 饱和度 (saturation) | 45 | 核心参数，营造极色彩风格 |
| | 亮度 (brightness) | 15 | 提升画面明亮度 |
| | 色相偏移 (hueShift) | 10 | 偏向洋红，强化梦幻感 |
